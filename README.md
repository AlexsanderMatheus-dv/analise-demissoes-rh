# Análise de Demissões e Retenção de Colaboradores

## Contexto
Este projeto tem como objetivo analisar dados de desligamentos de colaboradores, identificando padrões, causas e possíveis impactos relacionados à liderança, departamentos e perfil dos funcionários.

## Objetivo
Ajudar a área de Recursos Humanos a compreender os principais fatores que influenciam as demissões e apoiar a tomada de decisões estratégicas.

## Ferramentas Utilizadas
- Power BI
- Excel
- Dados simulados para fins de estudo

## Principais Análises
- Total de demissões
- Demissões voluntárias e involuntárias
- Demissões por departamento
- Motivos de desligamento
- Perfil etário dos colaboradores
- Demissões por gestor
- Evolução das demissões ao longo do tempo

## Principais Insights
- A maior parte das demissões é do tipo voluntária.
- Alguns departamentos apresentam taxas de desligamento acima da média.
- Há concentração de desligamentos em colaboradores com menor tempo de empresa.
- Determinados gestores apresentam maior número de desligamentos em suas equipes.

## Conclusão
Os dados indicam oportunidades de melhoria relacionadas à gestão de pessoas, clima organizacional e estratégias de retenção de talentos.
